# Launch Checklist

1. Set real Shopify Storefront API token.
2. Set real Shopify store domain.
3. Set real WhatsApp number.
4. Set real support phone and email.
5. Replace placeholder product images.
6. Import or create real products.
7. Create required Shopify collections.
8. Add products to collections.
9. Configure Cash on Delivery.
10. Configure Bank Transfer.
11. Configure JazzCash manual instructions.
12. Configure Easypaisa manual instructions.
13. Configure Pakistan shipping zones.
14. Test product pages.
15. Test Add to Cart.
16. Test cart quantity update.
17. Test remove from cart.
18. Test Shopify checkout.
19. Test WhatsApp product messages.
20. Test WhatsApp cart confirmation.
21. Test search.
22. Test mobile header and menu.
23. Set GA4 ID if needed.
24. Set Meta Pixel ID if needed.
25. Check privacy and terms copy.
26. Open Vercel `vercel.app` preview URL.
27. Update `NEXT_PUBLIC_SITE_URL`.
28. Bind custom domain when ready.
29. Confirm SSL is active.
30. Place a real test order before ads launch.
