# Product Upload Guide

## CSV import

Use `products/shopify-products.csv` to import 30 starter products for Pakistan home appliances and daily essentials.

## Add or edit products manually

1. Open Shopify Admin > Products.
2. Add or edit product title, description, photos, price, compare-at price, SKU, inventory, tags, and product type.
3. Set status to Active.
4. Add the product to a collection.
5. Save and test the product page on the Vercel site.

## Image requirements

The starter CSV uses placeholder image URLs. Replace all placeholder images with real product images before launch.

Recommended image format:

1. Square product photos.
2. At least 1000 x 1000 px.
3. Clean white or light background.
4. Clear view of product features and accessories.

## SEO

For each product, edit:

1. Page title.
2. Meta description.
3. URL handle.
4. Image alt text.

Use natural Pakistan shopping terms where relevant, such as COD, delivery across Pakistan, and product type.
